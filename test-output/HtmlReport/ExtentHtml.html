    check_circle cancel redo clear
    Validates and evaluates the account in the context of direct debit payment. Allows to submit IBAN and check if it’s valid. Method has 2-step validation:\n - length based validation, bank account considered to be valid if its length is between 7 and 34 chars, 400.00x error returned if validation fails.\n - external validation. 200(OK) with non empty riskCheckMessages returned if validation fails, otherwise isValid = true returned.\n - Validation works only for accounts written in IBAN format. Validation supports IBANs from the countries Austria, Germany, Switzerland, Finland, Sweden, Norway, and Denmark. Response message with error code 200.909 is returned If country is not supported.
